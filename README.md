# AI-vs-Human-Text-Classification
This Project takes as input Text and classifies it as either AI or Human. It starts with Data Understanding, Pre-Processing, Cleaning to make the data as neat as possible. 3 Models are experimented with ( Logistic Regression, Random Forest &amp; XGBoost). I have also put in place HyperparameterTuning using RandomizedSearchCV and testing 4 overfitting.
